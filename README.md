О программе:

===*Реализованно :*===

◉ Простой интуитивно-понятный интерфейс в Forms.

◉ Захват картинки при помощи EMGU.CV (Исключительно для захвата картинки).

◉ Комбинированный фильтр для обработки картинки, включающий в себя grayscale,thresholding,dilate фильтры.

◉ Динамическое управление реперами.

◉ Поиск в обозначенной реперами зоне цели и отрисовкой центра цели.

◉ Вывод координат на экран пользовательского интерфейса,координаты не изменяются при одинаковых вводных.

◉ Установлены необходимые библиотеки для возможной работы на "чистом" устройстве с ОС Windows 10.

◉ Выбор доступных камер для использования.

◉ Выбор формата репера на начальном этапе работы с пользовательским интерфейсом.
