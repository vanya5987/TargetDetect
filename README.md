О программе:

◉ Реализован простой интуитивно-понятный интерфейс в Forms.
◉ Реализован захват картинки при помощи EMGU.CV (Исключительно для захвата картинки).
◉ Реализован комбинированный фильтр для обработки картинки включающий в себя grayscale,thresholding,dilate.
◉ Реализовно динамическое управление риперами.
◉ Реализован поиск в обозначеной риперами зоне цели, и отрисовкой центра цели.
◉ Координаты цели программа выводит на экран пользовательского интерфейса,координаты неизменяются при одинаковых вводных.
◉ Установлены необходимые библиотеки для возможной работы на "чистом" устройстве с ОС Windows 10.
◉ Реализован выбор доступных камер для использования.
◉ Реализован выбор формата рипера на начальном этапе работы с пользовательским интерфейсом.
